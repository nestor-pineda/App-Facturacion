# Herramientas y paquetes del proyecto

Listado de dependencias npm y stack asociado. Cada ítem: nombre entre comillas, dos puntos, descripción en una línea.

## Infraestructura y runtime (implícitos o habituales)

- "Node.js" : "Entorno de ejecución del backend y de las herramientas de build/test."
- "PostgreSQL" : "Base de datos relacional usada vía Prisma y el driver pg."

## Raíz del monorepo (`package.json`)

- "concurrently" : "Ejecuta varios scripts npm en paralelo (p. ej. backend y frontend a la vez)."
- "@vercel/analytics" : "Telemetría y analítica web en despliegues compatibles con Vercel."

- "@types/nodemailer" : "Tipos TypeScript para nodemailer."
